FAQ

Foire Aux Questions

Comment fonctionne la compensation entre modules ?

Si, vers la fin de l'année scolaire, un étudiant a des modules non validés, mais que le nombre de modules non validés ne dépasse pas 3(en première année), il est possible de compenser l'un de ces modules non validés avec un autre module déjà validé.

Quand se passe la periode des examens?

Les examens du premier semestre se déroulent au mois de janvier, tandis que ceux du deuxième semestre ont lieu au mois de mai. Pour les étudiants ayant besoin de rattraper certains examens, une session de rattrapages est prévue 15 jours après la fin des examens semestriels.

L'abscence a-t-elle des impacts ?

Concernant l'abscence si vous depasser 4h vous aurrez un 1er avertissent, 8h d'abscence d'abscence vous serez blamer et si vous depasser 16h vous serez blamer pour la deusieme fois (cad vous n'aurrez pas le droit de passer le rattrapage ).

Comment le rattrapage est noté ?

Si un étudiant doit passer un examen de rattrapage, la note obtenue après cet examen de rattrapage ne prend pas en compte la note du contrôle continu ni celle de l'assiduité. De plus, cette note de rattrapage est plafonnée à 12/20 par module.

Comment fonctionne la validation des modules en 1ère année ?

Un module est validé si la note finale obtenue est supérieure à 12. La note finale d'un module est calculée à partir des notes des deux éléments qui le composent. Chaque élément de module est évalué selon trois critères : la note du contrôle continu, la note de l'examen final et la note d'assiduité. Les pourcentages attribués à chacun de ces critères sont définis par les professeurs.

Système d'éducation dans l'école ?

Après que le professeur a couvert 70% du contenu d'un élément de module, un contrôle continu est organisé ou un projet est réalisé à la place. Puis, une fois l'élément de module entièrement enseigné, un examen final est planifié pendant la période des examens semestriels.

Quel éditeur de code utiliser en 1ère année ?

La majorité des étudiants utilisent l'éditeur de code Visual Studio Code.
